[일반 MONO]
  /                        → 메인 (로그인 여부에 따라 /login 또는 /interpret 리다이렉트)
  /login                   → 로그인
  /setup                   → 초기 설정 (이름 + 언어)
  /interpret               → 통역 시작 (호스트 QR 생성)
  /home                    → 대화방 목록
  /contacts                → 연락처
  /settings                → 설정
  /join/:roomId            → 게스트 입장 (QR 스캔)
  /room/:roomId            → 채팅방
  /cs-chat                 → 고객센터 채팅
  /terms                   → 이용약관
  /privacy                 → 개인정보처리방침

[병원 모드 - 키오스크/직원 구조]
  /hospital?mode=kiosk&dept=reception      → 태블릿 키오스크 (QR만 표시)
  /hospital?mode=staff&dept=reception      → 직원 PC (환자 대기 목록 + 통역 시작)
  /hospital/join/reception                 → 환자 QR 스캔 후 입장 (언어 선택 → 채팅방)

[병원 모드 - 1:1 상담 (성형외과)]
  /hospital?mode=normal&dept=plastic_surgery  → 1:1 통역 (호스트 QR 생성, 환자 스캔)

[병원 모드 - 성형/피부 클리닉 전용 랜딩]
  /hospital/aesthetic                      → 랜딩 페이지 (키오스크 / 상담실 선택)
  /hospital/kiosk/plastic_surgery          → 성형외과 키오스크 (QR만 표시)

[병원 관리]
  /hospital/records                        → 병원 기록 조회
  /hospital-dashboard                      → 병원 대시보드

[기타]
  /kiosk                                   → 키오스크 (범용)
